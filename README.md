## Тема: Упрощение выражений, выполнив вычитание

### Описание задачи

Разработать программу на языке C для упрощения арифметических выражений. Программа должна принимать на вход выражение в виде строки, строить из него абстрактное синтаксическое дерево (AST), выполнять упрощения и выводить результат в двух форматах: текстовом и в виде древовидной структуры.

**Ключевые правила упрощения для варианта №2:**
1.  **Свертка констант:** `3 - 5` → `-2`
2.  **Перегруппировка:** `5 + b - 3` → `2 + b` (и аналогичные случаи, например `(x + 10) - 2` → `x + 8`)

### Реализация

Программа написана на чистом C и разделена на три логических модуля для наглядности.

1.  **Структура данных (`ast.h`)**
    -   Узел дерева представлен структурой `Node`.
    -   Для различения типов узлов (число, переменная, оператор) используется `enum NodeType`.
    -   Для эффективного использования памяти применяется `union data`, хранящий значение, имя или оператор.

2.  **Парсер (`parser.c`)**
    -   Использует алгоритм **Shunting-yard (Сортировочная станция)** для преобразования инфиксной строки в AST.
    -   Для работы алгоритма реализованы простые стеки на базе массивов.
    -   Содержит функцию `print_tree()` для вывода дерева в консоль в виде псевдографики.

3.  **Модуль упрощения (`simplifier.c`)**
    -   Функция `simplify(Node** node_ptr)` выполняет рекурсивный обход дерева (post-order).
    -   Использование **указателя на указатель (`Node**`)** позволяет изменять саму структуру дерева, а не только содержимое узлов.
    -   Реализованы все требуемые правила упрощения, включая свертку констант и перегруппировку с изменением структуры дерева.
